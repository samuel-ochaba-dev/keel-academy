# Keelacademy

An online school that turns developers who can make things work into engineers who can make things last.

## What students build

**Kairo**: a multi-service LLM application platform (streaming chat, visual workflow engine, RAG pipeline, model provider abstraction, tool system). Six services, thirteen weeks, deployed with one command.

## Curriculum layers

- **The Novel**: A junior engineer builds Kairo, makes real mistakes, gets corrected by a senior. Concepts taught through consequence.
- **The Build-Along**: Specs and test suites per chapter. No walkthroughs. Students write the implementation.
- **The Engineering Lexicon**: Production vocabulary explained in plain language with Kairo-specific examples.
- **DSA**: Every algorithm taught because the project requires it. Not abstract. Not disconnected.
