# Sliding Window

Sliding window is a DSA pattern for problems that ask about a contiguous range of values.

Instead of recalculating everything from scratch, you keep a moving window and update the answer as the range expands or contracts.

Why include this in M0? Because Keel Academy is not a docs site with vibes. The platform has to prove the content model can carry engineering concepts and DSA concepts side by side from day one.

A good first mental model:

1. add a new item to the window,
2. remove the old item when constraints break,
3. update the best answer,
4. repeat.
